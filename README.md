# 🧭 Threat Compass

> Real-time global vulnerability tracker with MITRE ATT&CK mapping & mitigation intelligence.

## What is Threat Compass?

Threat Compass is an open-source cybersecurity dashboard that tracks the latest globally reported vulnerabilities in real-time. It maintains a complete historical record of past threats and provides actionable mitigation strategies — all mapped to the MITRE ATT&CK framework.

Built for SOC analysts, threat hunters, and blue teamers who need situational awareness fast.

## Features

- 🔴 **Live Vulnerability Feed** — latest CVEs and global threats updated in real-time
- 🗂️ **Historical Records** — full archive of past vulnerabilities with severity ratings
- 🛡️ **Mitigation Guidance** — actionable steps for each vulnerability
- 🎯 **MITRE ATT&CK Mapping** — every threat mapped to tactics & techniques
- ⚡ **Fast & Clean UI** — built for analysts, not managers

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn-ui
- Supabase

## Getting Started
```sh
git clone https://github.com/Vikram-Shukl/threat-compass
cd threat-compass
npm i
npm run dev
```
